# DefElt Студент-Ученик программа

## Описание программы
При запуске компьютера открывается программа, требующая логин и пароль. При корретном входе в аккаунт если это аккаунт ученика, то программы скрывается и анализирует запущенные процессы на компьютере и отправляет на сервер. Если это аккаунт учителя, тогда открывается специальное меню, где можно: 
 * Смотреть кто пришел на занятие и во сколько.
 * Создавать/Просматривать занятие конкретной группы.
 * Просматривать какие программы запущенны у ученика.
 * Закрывать выбранную программу у ученика.
 * Создавать группы.
 * Регистрировать новых учеников

 ## Установка

В папке с установкой есть сервер и клиент. Клиент не запустится без сервера!
